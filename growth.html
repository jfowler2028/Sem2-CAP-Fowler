Growth

For a project that represents personal growth, I chose the Unit 6 turtle programming assignment. This work represents an area of growth because I first struggled with using turtle programming, but I made it work, and I was super proud of the result. The hardest area for me to master when learning turtle programming and completing this project was making sure the turtle was going in the right direction and making sure my design didn't overlap with other things. One obstacle I faced when completing this project was making the waves in my ocean scene, because the intricate pattern that I had to replicate was difficult to repeat and make the same every time, while still ending up in the same spot and not going off the page. My level of effort in this class directly influenced my success because when I was struggling, I took time out of my day to come to office hours for help, and I utilized all the in-class time to complete the assignment I think I would add more detail, knowing what I know now, because my knowledge of coding has improved so much, and I believe now I would be able to create a more detailed version of this project.
